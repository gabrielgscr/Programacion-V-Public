# Programacion-V-Public

Repositorio oficial de apoyo para el curso de Programación V del CUC.

Este curso es dirigido y administrado por Gabriel González Solano.

## Bienvenida 🚀

Este espacio fue creado para aprender programando en serio, con proyectos que se parecen a escenarios reales del mundo tech.
La idea es avanzar paso a paso, construir productos funcionales y fortalecer tu perfil como desarrollador o desarrolladora.

## Descripción del curso 📘

Programación V es un espacio orientado al desarrollo de competencias avanzadas de programación, con enfoque en:

- Programación Web para el desarrollo de aplicaciones modernas.
- Diseño e implementación de APIs para integración de servicios.
- Arquitecturas basadas en microservicios.
- Desarrollo móvil con Android nativo y enfoques multiplataforma.
- Buenas prácticas de codificación, mantenibilidad y trabajo colaborativo.

## Objetivo general 🎯

Fortalecer la capacidad del estudiante para analizar, diseñar, implementar y documentar soluciones de software integrales, conectando aplicaciones web, APIs, microservicios y clientes móviles bajo principios modernos de desarrollo.

## Características relevantes del curso ⭐

- Enfoque práctico con ejercicios guiados y retos progresivos.
- Integración de teoría y práctica en cada unidad.
- Uso de repositorios para el seguimiento de actividades y evidencias.
- Desarrollo de habilidades de depuración, refactorización y documentación.
- Énfasis en integración entre frontend web, backend API y aplicaciones móviles.
- Aplicación de principios de arquitectura para soluciones escalables con microservicios.

## Competencias que se trabajan 🧠

- Diseño de soluciones Web con separación por capas y componentes.
- Construcción y consumo de APIs REST con validaciones y manejo de errores.
- Aplicación de principios de microservicios para desacoplar dominios y responsabilidades.
- Desarrollo móvil en Android nativo y en frameworks multiplataforma.
- Integración de servicios backend con clientes web y móviles.
- Trabajo en equipo con control de versiones y documentación técnica.

## Metodología 🛠️

El curso se desarrolla con una metodología activa:

- Clases explicativas con ejemplos de código.
- Talleres prácticos en clase.
- Actividades autónomas y entregables por unidad.
- Retroalimentación continua sobre soluciones propuestas.

## Experiencia de aprendizaje ⚡

Durante el curso trabajaremos con una dinámica orientada a jóvenes, enfocada en aprender haciendo:

- Mini retos semanales para practicar habilidades clave.
- Laboratorios por equipo con roles de trabajo colaborativo.
- Casos prácticos conectados con apps y servicios actuales.
- Proyecto final con enfoque tipo portafolio para mostrar tus capacidades.

## Contenido del curso por unidades 🗂️

1. 🌐 Programación Web moderna.
   Desarrollo de aplicaciones web con arquitectura por capas, manejo de formularios, rutas, autenticación y buenas prácticas de frontend y backend.
2. 🔌 Diseño y desarrollo de APIs.
   Creación de APIs REST, definición de endpoints, validación de datos, manejo de errores, documentación y versionado de servicios.
3. 🧩 Microservicios y comunicación entre servicios.
   Principios de arquitectura de microservicios, desacoplamiento, responsabilidades por dominio, comunicación síncrona/asíncrona y despliegue básico.
4. 🤖 Programación móvil Android nativa.
   Fundamentos de desarrollo Android nativo, ciclo de vida de actividades, navegación, consumo de APIs y persistencia local.
5. 📱 Programación móvil multiplataforma.
   Desarrollo de aplicaciones para múltiples plataformas desde una sola base de código, integración con APIs, manejo de estado y estructura del proyecto.
6. 🏁 Proyecto integrador.
   Construcción de una solución completa que conecte aplicación web, API, microservicios y cliente móvil, aplicando estándares de calidad y documentación técnica.
